# 🎤 60 秒即兴 · Impromptu 60

练**临场表达**（即兴技能）的小工具：抽一个题 → 张口说满 60 秒。中英双语题库，顺便练英语口语。和吉他工具互补——吉他是运动/封闭/程序性技能，即兴是认知/开放技能。

### 🎤 [在线使用 → jada-q.github.io/impromptu-speak](https://jada-q.github.io/impromptu-speak/)

## 功能
- **40 道双语即兴题**，随机抽，可切 双语 / English / 中文
- **60 秒计时**：最后 10 秒变黄提示 + 倒数滴答，结束有完成音
- **PREP 结构提示**（观点→理由→例子→重申），不知道怎么说时照着搭
- 可选「先想 15 秒」准备模式
- **计数**：今天练了几次 + 累计（每天 3 次达标 🎉）
- 单文件 HTML + Web Audio 提示音，零素材；PWA 可装到手机主屏离线用

## 怎么练
每天 3 次。**说满 60 秒不卡壳 = 胜利**，内容好不好不重要——练的是"开口的流畅度"，这是程序性技能，靠重复。
